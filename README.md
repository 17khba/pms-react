## 技术栈：

react + webpack + react-router + antd + axios + less + ES6

## 运行项目

```shell
# 克隆本项目
git clone https://github.com/17khba/pms-react.git

# 进入项目根目录
cd pms-react

# 安装依赖
npm/cnpm i

# 本地运行
npm start

# 打包
npm run build
```
